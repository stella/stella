# Mobile

The mobile workspace is an Expo app for iOS, Android, and local web previews.
It uses Expo Router with typed routes and React Compiler enabled.

## Development

From the repository root:

```sh
bun --filter @stll/mobile ios
bun --filter @stll/mobile android
bun --filter @stll/mobile web
```

Run the native compatibility and repository checks with:

```sh
bun --filter @stll/mobile doctor
bun --filter @stll/mobile typecheck
bun --filter @stll/mobile lint
bun --filter @stll/mobile build
```

The app intentionally shares the repository's React 19 patch release. React
Native 0.86 supports that range, and using one copy prevents Metro and native
autolinking from seeing duplicate React runtimes. Expo's exact bundled React
pin is therefore excluded from its package-version comparison; all native
module versions remain checked.

Expo Metro requires Babel 7 while the web build uses Babel 8. Both workspaces
declare their required major directly, and the workspace-hygiene check ratchets
that deliberate split while Sherif continues checking every other dependency.

Native `ios/` and `android/` directories are generated by Expo prebuild and
stay uncommitted until a capability requires maintained native source.
