stella is designed to be self-hosted. You retain full control over your data and infrastructure.

## Docker

The recommended way to self-host stella is with Docker Compose.

```bash
docker compose up -d
```

Details coming soon.

## Requirements

- PostgreSQL 16+
- S3-compatible object storage (AWS S3, MinIO, Cloudflare R2)
- 2 GB RAM minimum
