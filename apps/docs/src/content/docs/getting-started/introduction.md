stella is an open-source legal workspace that brings documents, matters, review, and research into one place.

## Key features

- **Document management** — upload, organize, and search legal documents
- **Matter tracking** — manage cases, clients, and deadlines
- **AI-powered review** — extract key fields from large document sets into sortable tables
- **Template drafting** — turn structured inputs into clean first drafts
- **Grounded research** — search your workspace and trusted sources with cited answers

## Deployment options

- **Cloud** — managed hosting at stll.app
- **Self-hosted** — run on your own infrastructure with full control

## Open source

stella is licensed under Apache-2.0. The source code is available on [GitHub](https://github.com/stella/stella).
