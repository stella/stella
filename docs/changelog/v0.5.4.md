# Right-to-left editing and Arabic localization

The document editor now auto-detects right-to-left base direction, so Arabic and
Hebrew paragraphs render and export to Word correctly without manual tagging.
Arabic interface terminology was revised for accuracy and consistency, and
localized "New {layout}" view names now agree with each language's grammatical
gender.

Web search and URL fetch can use per-workspace API keys, and the desktop app
supports linking to a self-hosted workspace.
