# Anonymization

## Hide selected information before it's sent to external models or tools

Open a document and the inspector now lists the terms the on-device pipeline picked up — grouped by category. Disagree with a match? One click takes it off the list, and the change scopes to this document, this workspace, or (under firm-admin settings) the whole firm. The same overrides apply when chat reaches an external model and when MCP tools return text.
