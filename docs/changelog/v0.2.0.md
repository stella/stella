# Anonymization

## Hide selected information before it's sent to external models or tools

stella can now, optionally, take a first pass at the documents and chats you're about to send and mask names, dates, company details and similar information. When the response comes back, stella swaps the real values in again. You can see what's being masked and change it any time.
