# Stella is now connected to your other tools

Stella is now fully ready to be controlled by and inform your other systems.
This builds on our vision that everything in stella is controllable by humans
and machines alike, as we see an increasing need to blend the two in legal work.

In addition, document review grows with a redesigned suggestion flow and
exact-passage citation highlights.

At the administrative level, accounts can enable two-factor authentication
(TOTP with backup codes), and administrators can inspect user activity logs.

Once again, this release benefited from ideas and code contributed by the wider
community.
