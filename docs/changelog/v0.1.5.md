# MCP polish

## Tighter knowledge page, cleaner connect flow, per-firm tool toggles.

The MCP knowledge page is now a single-lane layout: brand, URL, and a use-in-chat toggle per server. Adding a new server happens inline — paste the URL, no modal. Owners and admins can disable a built-in tool (like ARES) for the whole firm with one switch. The OAuth popup closes itself when the connection completes, and the parent tab refreshes automatically.

Plus better external PDF previews, more honest error toasts in the inspector, and various small fixes around the app.

Keep the feedback coming.
