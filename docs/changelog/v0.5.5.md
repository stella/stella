# Search, desktop document handling, and release hardening

This patch improves Arabic search normalization across workspace, contact,
entity, corpus, and chat surfaces. It also tightens document handling in the
desktop app by validating DOCX downloads and preserving the expected `.docx`
extension.

The release includes smaller fixes for chat compaction, metered chat streams,
template permissions, view-menu interactions, and desktop session resilience.
Release and CI workflows were hardened around tag creation, publish dispatch,
release notes, container credentials, and registry build behavior.
