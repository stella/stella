# Export your tables

## Pull table results out of stella into a spreadsheet

Export the values from a table as CSV or XLSX, ready for Excel, Google Sheets, or anywhere else you work with data. Reviewed and follow-up flags carry over as cell colours, so you can keep going  
 where you left off.

Plus several bugfixes around the app.
