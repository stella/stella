# NDA checklist

<!-- guide: Replace these example rows with your own rules. Keep the shape: a stable code, one check per row, and a concrete threshold — "looks unusual" is not testable, "longer than 5 years" is. -->

| Code | Check                                  | Flag when…                                                                                      |
| ---- | -------------------------------------- | ----------------------------------------------------------------------------------------------- |
| C1   | Definition of confidential information | it covers all exchanged information with no carve-outs (public domain, independently developed) |
| C2   | Term of confidentiality                | longer than 5 years, or indefinite for information that is not a trade secret                   |
| C3   | Governing law                          | missing, or different from what the parties agreed elsewhere in the deal                        |
| C4   | Dispute forum                          | missing, or exclusive in a forum unrelated to either party                                      |
| C5   | Mutuality                              | obligations bind only one party — see Classification in SKILL.md                                |
| C6   | Hidden restraints                      | a non-solicitation or non-compete clause is embedded in the NDA                                 |
| C7   | Contractual penalty                    | a fixed penalty per breach with no cap, or out of proportion to the foreseeable harm            |
| C8   | Return or destruction                  | no obligation to return or destroy received materials when the agreement ends                   |
