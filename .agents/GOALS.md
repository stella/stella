# Goals

High-level priorities for Stella. Updated as the project evolves.

## Current Focus

1. **Core DMS** — matters, documents, search. Free forever. Must be
   reliable and fast before anything else.
2. **AI Review** — tabular review for due diligence and discovery.
   Paid by usage (credits).
3. **AI Research** — legal research agent with external source access.
   Paid by usage (credits).
4. **Auth** — better-auth with organisation support, roles
   (Owner/Admin/Member), and email OTP.

## Next

- Document management (full-text search, versioning)
- Matter lifecycle (open/closed, archival to Glacier)
- Automation blocks (generic primitive: string | () => string)
- Billing integration (Stripe, credit packs, boosters)

## Non-Goals (for now)

- Word/Outlook plugin (parked)
